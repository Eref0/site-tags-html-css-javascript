Tags, abreviações chave e utilitários

    O que é a tag?

    R: tag é o elemento fundamental do HTML, e está representado pelo conteúdo que situa-se entre o sinal "<" e ">". A função da tag é indicar ao programa que está lendo o texto qual é a função que deverá ser executada pelo texto, sendo assim ela está contida no HTML das páginas mas não aparece no corpo do site, ou seja, os internautas que acessam o site não conseguem visualizar as tag, mas é graças a elas que é possível visualizar este texto de definição.

    A tag é iniciada em sua abertura: "< li >", por exemplo, e termina em seu fechamento "< /li >". Note que no fechamento da tag, na qual vamos inserir algo dentro, temos o elemento "/", este é o indicativo do fechamento de tag.

    Algumas tags básicas:

  • - tag header: a tag "< header> < /header>" representa o cabeçalho da página, a porção superior onde geralmente ficam os links para outras páginas ou login.
  • - tag footer: a tag "< footer> < /footer>" é o nosso rodapé, a porção inferior da página. Nesta região geralmente se encontra os links para ajuda, sobre, redes sociais, etc.
  • - tag main: a tag "< main> < /main>" é utilizada para se escrever os principais elementos que estarão contidos no corpo (body) de uma página.
  • - tag tittle: a tag "< title > texto desejado aqui < /title >" é utilizada para dar título a uma página e geralmente ela está presente no head da página. Lembrando que o head nada mais é do que a região da página que não precisa receber conteúdo e encontra-se antes do body da página, que é a região corpo da página.
  • - tag body: a tag "< body > escreva e adicione elementos a sua página aqui < /body >" é o corpo de uma página, é nesta tag que estará contido todos os elementos visuais da página. É onde você está lendo esta tag neste exato momento, por exemplo.
  • - tag li: quando o li encontra-se entre "> li <" ela possibilida a criação de uma lista não ordenada. Como esta que estamos criando agora, para a definição das tags. Essa rag é utilizada da seguinte forma: abertura da tag "< li >" texto ou comandos que vserão aplicados "< /li >" fechamento da tag.
  • - tag p: a tag é o elemento de texto, como a definição da tag li e da tag p. E como qualquer outra tag, o p deve estar contido entre '< e>', exemplo: "< p>" (abertura do elemento texto) e "< /p>" (fechamento to elemento texto).
  • - tag ul: a tag "< ul> < /ul>" é utilizada para definir um lista li sem uma ordem numérica.
  • - tag h1 ao h6: a tag "< h1> < /h1>" é utilizada da mesma forma que a tag "< p> < /p>", com a diferença que o texto inserido dentro da tag terá função de título e não parágrafo. Podemosobservar a utilização desta tag no título desta página "Lista de Abreviações e Conceitos Chave para HTMLS e CSS, por: Eref". Já as tags que seguem sequência tem valores menores de acordo com a ordem crescente numérica, por exemplo: a tag "< h2> < /h2>" terá valor menor que a tag "< h1> < /h1>" e valor maior que as tags h3, h4, h5 e h6.
  • - tag div: a tag "< div> < /div>" é uma tag utilizada para dividir uma porção do HTML da página. Essa tag, quando utilizada sozinha, de certa forma não representa nada, mas pode ser utilizada para agrupar elementos para criação de um estilo, como os elementos "class" e "id" por exemplo.
  • - tag img: a tag "< img src="" alt="">" é utilizada para colocar uma imagem ou referenciar que uma imagem ocorre no projeto, também é possível linkar uma imagem utilizando esta tag. Observamos que existe mais que o elemento img representado nesta tag. Vamos entender estes elementos:

    • O elemento "src=" representa o diretório onde a imagem está, ou seja, o endereço da imagem no seu computador ou seu URL na web.

    O elemento "alt=" possibilita que a descrição desta imagem ocorra via HTML. Desta forma, caso a imagem não seja exibida, a descrição aparece em seu lugar. Também vale ressaltar que pessoas com deficiência visual não conseguem enxergar as imagens presentes nas páginas, mas com uma boa descrição da imagem é possível tornar a página acessível para programas que auxiliam na leitura de imagens.

  • - tag a: para utilização de links basta inserir a tag "< a href="link desejado aqui"> texto desejado aqui < /a >", desta maneira voce terá um link que poderá ser estilizado no CSS e/ou JavaScript depois.
  • - tag button: a tag "< button > < /button >" serve para adicionar um botão à página que você está criando. Porém esta tag adiciona apenas um retângulo com um espaço onde você pode incluir um texto. Para personalizar esse botão são utilizadas outras tags em conjunto com o CSS ou JavaScript.
  • - tag section: a tag "< section> < /section>" é utilizada para agrupar tags. Esta tag é utilizada para definir uma sessão da página, onde os elementos contidos dentro da tag compõem esta sessão.
